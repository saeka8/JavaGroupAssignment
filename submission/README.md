# QuizPlatform - Submission Package

## How to Run

### Option 1: Double-click (Windows)
Simply double-click `RUN.bat`

### Option 2: Command Line
```
java -jar QuizPlatform-1.0-SNAPSHOT.jar
```

## Requirements
- Java 18 or higher installed
- All files in this folder must stay together
